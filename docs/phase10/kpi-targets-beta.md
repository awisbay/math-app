# Phase 10 KPI Targets (Beta)

## Product Stability KPIs
- Crash-free sessions rate.
- API error rate for critical endpoints.
- Session generation success rate.

## Engagement KPIs
- Session completion rate.
- Average sessions per active user.
- Daily active streak users.

## Retention KPIs
- D1 and D7 retention.
- Return rate after first completed session.

## Content KPIs
- Question report rate (quality issues).
- Post-publish correction rate.

## Ops KPIs
- Mean time to acknowledge incidents.
- Mean time to resolve critical issues.
- Patch release turnaround time.

## Notes
- Final numeric thresholds should follow Phase 8 quality gate baselines.
- Track trends by grade band to identify segment-specific issues.
