# Phase 8 KPI Dashboard Specification

## Product Dashboard
Key metrics:
- DAU/WAU/MAU.
- Session starts per day.
- Session completion rate.
- Average score by grade.
- Average time-to-complete session.

## Retention Dashboard
- D1/D7/D30 retention.
- Streak active users.
- Milestone distribution.
- Reminder open rate.

## Reliability Dashboard
- API success/error rates.
- P95 latency by endpoint.
- Crash-free session rate.
- Queue processing health.

## Content Quality Dashboard
- Validation pass rate.
- Moderation rejection rate by reason.
- Post-publish correction rate.
- Topic coverage completeness by grade.

## Release Health Dashboard
- Open bugs by severity.
- Regression count by release.
- Test pass rate trend.
- Build stability trend.
